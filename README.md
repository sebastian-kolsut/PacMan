

# instructions for creating a pull request

1. Commit na twojego brancha

```bash
git branch branch-name
git switch branch-name # przenosi cie na branch
```
2. Push

```bash
git push -u origin nazwa-twojego-brancha
```

3. Wchodzisz sobie na ten adres wpisujac nazwe swojego brancha

```https
https://github.com/sebastian-kolsut/PacMan/pull/new/nazwa-brancha
```
I tam juz sobie wszystko wypelniach co trzeba.


Work Flow:
git switch main
git fetch origin
git pull
